## 1.사용기술
1.Language: Python, HTML5, CSS3, Javascript  
2.Database: MySQL  
3.Server: Nginx, Gunicorn, Django  
4.Development Environment: Windows 10, Ubuntu Linux  
5.Etc: MySQL Workbench, VS code, Anaconda, Jupyter notebook, VMware
